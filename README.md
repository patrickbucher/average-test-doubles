# Setup

Optional: Install [Deno Extension](https://open-vsx.org/vscode/item?itemName=denoland.vscode-deno) for Visual Studio Code.

# Execution

Run it:

    deno run --allow-read demo.ts mean 1-to-10.txt
    deno run --allow-read demo.ts median 1-to-10.txt
    deno run --allow-read demo.ts mode 1-to-10.txt
